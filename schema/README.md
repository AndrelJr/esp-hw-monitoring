SCHEMATIC
==========

There are thre components (see illustrations):
 * ESP8266 esp12e.
 * 240*320 2.8 SPI LCD with ILI9341 and XPT2046 touch screen.
 * DHT22 temperature module (with resistor).
 
LCD and touch screen are controlled by the arduino with SPI.
